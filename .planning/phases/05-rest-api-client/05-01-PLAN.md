Create the core HTTP client with connection pooling and API key injection.

**base.py:**
- Create `OnCallHealthClient` class:
  ```python
  class OnCallHealthClient:
      def __init__(self, api_key: str, config: ClientConfig | None = None):
          self.api_key = api_key
          self.config = config or ClientConfig.from_env()
          self._client: httpx.AsyncClient | None = None
          self._created_at: float = 0
  ```

- Implement `_get_client() -> httpx.AsyncClient`:
  - Check if client needs recreation (age > max_client_age_seconds)
  - If recreation needed, call `_recreate_client()`
  - If no client, create with `_create_client()`
  - Return the client

- Implement `_create_client() -> httpx.AsyncClient`:
  - Create async event hook `inject_api_key(request)` that adds `X-API-Key` header
  - Create `log_response(response)` hook that logs method, path, status at DEBUG level
  - Return `httpx.AsyncClient` with:
    - `base_url=self.config.base_url`
    - `limits=self.config.to_httpx_limits()`
    - `timeout=self.config.to_httpx_timeout()`
    - `event_hooks={"request": [inject_api_key], "response": [log_response]}`

- Implement `_recreate_client()`:
  - Store old client reference
  - Set `self._client = None`
  - Call `await old_client.aclose()` in try/except (log warning on error)

- Implement `request(method: str, path: str, **kwargs) -> httpx.Response`:
  - Get client via `_get_client()`
  - Make request: `response = await client.request(method, path, **kwargs)`
  - If `response.status_code >= 400`, call `map_http_error_to_mcp(response)` and raise result
  - Return response

- Implement convenience methods:
  - `get(path: str, **kwargs) -> httpx.Response`
  - `post(path: str, **kwargs) -> httpx.Response`
  - `put(path: str, **kwargs) -> httpx.Response`
  - `delete(path: str, **kwargs) -> httpx.Response`

- Implement `close()`:
  - If client exists, call `await self._client.aclose()`
  - Set `self._client = None`

- Implement async context manager (`__aenter__`, `__aexit__`) for proper cleanup
